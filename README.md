# TrackIt

A habits tracking app written in React that allows you to improve your discipline!

Try it out now at https://track-it-oktllrsjo-kethllen.vercel.app/

## About

This is a web application with which many people can manage their daily habits. Below are the implemented features:

- Sign up
- Log in
- List all registered user habits
- List all habitps of the day
- Track the progress of your performed habits
- Add habit
- Remove habit

## How to run

1. Clone this repository
2. Install dependencies
```bash
npm i
```
3. Run the front-end with
```bash
npm start
```
4. You can optionally build the project running
```bash
npm run build
```
5. Finally access http://localhost:3000 on your favorite browser (unless it is Internet Explorer. In this case, review your life decisions)
